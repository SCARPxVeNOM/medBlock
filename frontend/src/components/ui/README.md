# Magic UI Components

This directory contains Magic UI components integrated from [magicui.design](https://magicui.design/docs/components).

## Components

- **AnimatedGradientText** - Animated gradient text effect
- **MagicCard** - Interactive card with hover effects
- **ShimmerButton** - Animated button with shimmer effect
- **BorderBeam** - Animated border beam effect
- **BlurFade** - Fade-in animation with blur
- **AnimatedGridPattern** - Animated grid background pattern
- **NumberTicker** - Animated number counter

## Usage

All components are ready to use and styled with Tailwind CSS. They work with the dark theme gradient background used throughout the application.
